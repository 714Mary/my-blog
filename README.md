# 我的个人博客项目（v1.0）

这是一个基于 **前后端分离** 的个人博客项目，后端使用 SpringBoot，前端使用 Vue 实现。

---

## 项目功能
用户文章发布与展示
基础文章 CRUD 接口
响应式前端页面
本地开发环境可正常运行

---

## 技术栈
- **后端**：Spring Boot + MyBatis + MySQL
- **前端**：Vue + Vite
- **版本管理**：Git + GitHub

---

## 本地运行
### 后端
```bash
# 1. 导入项目到 IDEA
# 2. 配置 application.yml 数据库信息
# 3. 运行 BlogBackendApplication.java
```
### 前端
```bash
# 1. 进入前端目录
cd blog-front
# 2. 安装依赖
npm install
# 3. 启动开发服务
npm run dev
# 4. 打开浏览器访问本地地址即可
